## Prerequisites

| Requirement | Details |
|---|---|
| **Claude Subscription** | Claude Pro ($20/month), Max, or Teams. Free tier NOT supported. Sign up at [claude.ai](https://claude.ai) |
| **Operating System** | macOS 13.0+ or Windows 10+ |
| **Windows only** | [Git for Windows](https://git-scm.com/downloads/win) must be installed first |

This takes about 10 minutes. If you hit a snag, there's a troubleshooting section at the bottom.

## Step 1: Open Your Terminal

All the commands in this guide need to be typed into your Terminal (also called command line). Here's how to open it:

**Mac:**
1. Press `Cmd + Space` to open Spotlight
2. Type "Terminal"
3. Press Enter

**Windows:**
1. Press the Windows key
2. Type "PowerShell"
3. Press Enter

You'll see a window with a blinking cursor — this is where you'll type commands.

<Callout type="info">
**Windows users:** Make sure you have [Git for Windows](https://git-scm.com/downloads/win) installed before continuing. Claude Code needs it to run commands. You do not need to run PowerShell as Administrator.
</Callout>

## Step 2: Install Claude Code

In your terminal, run this command:

**Mac:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows PowerShell:**
```powershell
irm https://claude.ai/install.ps1 | iex
```

Wait for the installation to complete. You'll see a success message when it's done.

<Callout type="warning">
**Important:** After the install finishes, **completely close your terminal window and open a fresh one** before continuing. The installer adds Claude Code to your system PATH, but your current terminal won't see it until you restart it.
</Callout>

## Step 3: Verify Claude Code Is Installed

In your **new** terminal window, run this command:

```bash
claude --version
```

You should see a version number (e.g., `2.0.30`). If you do — Claude Code is installed.

<Callout type="warning">
**Still seeing `command not found`?** Make sure you opened a brand new terminal window (not a new tab in the old one). On Mac, quit Terminal entirely (`Cmd + Q`) and reopen it. On Windows, close PowerShell and open a new one from the Start menu.
</Callout>

## Step 4: Sign In

Launch Claude Code by typing this into your Terminal and press Enter:

```bash
claude
```

The first time you run it, Claude Code will ask you to sign in:

1. Select **"Claude account with subscription"** when prompted
2. A browser window will open automatically
3. Sign in with your Claude Pro or Max account
4. Come back to the Terminal — you should see Claude Code ready to go

Once signed in, run a quick health check from your terminal:

```bash
claude doctor
```

This verifies your installation, subscription, and system configuration. If everything passes, you're all set.

## Troubleshooting

| Problem | Solution |
|---|---|
| `claude: command not found` | Quit your terminal app entirely (Mac: `Cmd + Q`, Windows: close the window) and open a fresh one. If still failing, run the install command again in the new terminal |
| Browser doesn't open for sign-in | Copy the URL shown in the terminal and paste it into your browser manually |
| `claude doctor` shows errors | Run the install command again to update Claude Code |
| "No subscription found" error | Make sure you have a Claude Pro ($20/mo) or Max subscription at [claude.ai](https://claude.ai) |
| Windows: Claude Code can't find Git Bash | Install [Git for Windows](https://git-scm.com/downloads/win), then restart your terminal |

## Try It

Once Claude Code is running and you've signed in, try asking it something simple to confirm it works:

```
What day of the week is it today?
```

If Claude responds — congratulations, you're set up. Move on to **0.2: Start & Clone** to get your first real project going.
