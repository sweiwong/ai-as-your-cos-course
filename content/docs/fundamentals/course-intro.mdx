## Learning by Doing

Every lesson has two parts:

| Part | Where | What |
|------|-------|------|

## How Each Lesson Works

<Steps>
<Step>
### Read the lesson page (this website)

Each page explains a concept, shows examples, and gives you the context you need.
</Step>
<Step>
### Run the `/start` command (Claude Code)

Every lesson has a slash command like `/start-1-1` that launches guided exercises in your terminal. Claude walks you through practice tasks using real files in your course project.
</Step>
<Step>
### Check your work

Each exercise includes criteria so you can tell if it worked. 
</Step>
</Steps>
